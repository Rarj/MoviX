## Getting Started

### Prerequisites
- Java Development Kit (JDK) 17 or higher
- Gradle 8.2.X
- Android Studio Hedgehog | 2023.1.1 or higher

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Rarj/MoviX.git
   cd MoviX

2. Build Project
   ```sh
   ./gradlew build
   
3. To run the apps, use the following command
   ```sh
   ./gradlew run
